# Static Assets

## Network Diagrams

Place network topology images here:
- `topology.png` - Main network diagram
- `docker-network.png` - Container networking
- `security-zones.png` - Security architecture

## Creating Diagrams

Recommended tools:
- draw.io / diagrams.net
- Mermaid (code-based diagrams)
- Lucidchart

## Image Guidelines

- Format: PNG or SVG
- Resolution: Minimum 1920x1080 for network diagrams
- File size: Keep under 2MB each
- Naming: Use lowercase with hyphens
